## Lightpanda Browser 无头浏览器：为 AI 构建的原生网页浏览器

- Lightpanda 是一款从头为机器构建的 AI 原生网页浏览器，旨在实现快速、可扩展的网页自动化，且内存占用极小。

- 专为无头使用而设计：

  - 执行 JavaScript

  - 支持 Web API

  - 通过 CDP 与 Playwright、Puppeteer 兼容

- 为 AI 智能体、大语言模型训练、数据抓取和测试提供快速的网页自动化：

  - 超低内存占用（仅为 Chrome 的十分之一）

  - 极快的执行速度（比 Chrome 快 10 倍）

  - 即时启动
