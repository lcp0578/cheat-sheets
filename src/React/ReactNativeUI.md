## React Native UI component libraries
- [NativeBase: Mobile-first, accessible components for React Native & Web to build consistent UI across Android, iOS and Web.](https://github.com/GeekyAnts/NativeBase)
- [React Native Elements: As a cross platform UI Toolkit, you can now use RNE on the web & share your codebase between your React Native + React web apps.](https://github.com/react-native-elements/react-native-elements)
- [UI Kitten: React Native UI Library based on Eva Design System](https://github.com/akveo/react-native-ui-kitten)
- [React Native Paper: Material Design for React Native (Android & iOS)](https://github.com/callstack/react-native-paper)
- [Tamagui: Style React fast with 100% parity on React Native, an optional UI kit, and optimizing compiler.](https://github.com/tamagui/tamagui)
